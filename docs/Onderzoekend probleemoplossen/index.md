# Verdeling van onderzoeksonderwerpen

Dit zijn de onderwerpen waar we uit kunnen kiezen:

- **Interface List** met implementatieklassen `ArrayList`, `LinkedList`
- **Interface Set** met implementatieklassen `HashSet`, `TreeSet`
- **Interface Map** met implementatieklassen `HashMap`, `TreeMap`
- **Interface Queue** met implementatieklassen `LinkedList`, `ArrayDeque`
- **Interface Stream** en het gebruik ervan binnen klassen uit het Java Collection Framework
- **Functionele interfaces** en het gebruik ervan binnen klassen uit het Java Collection Framework
- Elk ander onderwerp van interesse voor een student dat **niet conflicteert** met bovenstaande lijst en wordt goedgekeurd door een **technisch docent**
